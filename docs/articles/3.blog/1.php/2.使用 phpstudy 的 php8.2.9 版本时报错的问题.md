# phpstudy 使用 php8.2.9 版本报错问题

### 错误
使用以下命令：
```shell
php -v
```
报错：
Fatal error: Directive 'track_errors' is no longer available in PHP in Unknown on line 0

### 解决方案
在D:/phpstudy_pro/Extensions/php/php8.2.9nts/php.ini中将track_errors=On改为track_errors=Off
